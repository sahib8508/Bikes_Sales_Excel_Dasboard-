# Bike Sales Dashboard Analysis

## About the Project

I created this project to practice the complete data analysis process in Excel, starting with raw data and ending with an interactive dashboard. The dataset contains information about customers and whether they purchased a bike. The goal was to clean the data, organize it for analysis, identify trends, and present the findings in a simple and interactive way.

---

## Workbook Structure

The workbook is divided into four sheets:

### 1. Raw Data

This sheet contains the original dataset before any modifications were made. It serves as the source of truth and allows the cleaning process to be traced back to the original records.

### 2. Cleaned Data

In this sheet, I prepared the data for analysis by:

* Removing duplicate records
* Checking for inconsistencies
* Creating additional fields using Excel formulas
* Organizing the dataset into a more analysis-friendly format

I also created age categories using IF formulas so that customers could be grouped into meaningful ranges rather than analyzing individual ages.

### 3. Pivot Tables & Charts

After cleaning the data, I used pivot tables to summarize information and answer questions such as:

* Which age groups are more likely to purchase bikes?
* How does income relate to bike purchases?
* Are there noticeable differences across customer segments?

The pivot tables were then converted into charts to make the results easier to understand.

### 4. Dashboard

The final sheet combines the key charts into a single dashboard. Slicers were added to make the dashboard interactive, allowing users to filter and explore the data from different perspectives without changing the underlying analysis.
